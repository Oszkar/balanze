Balanze / cross-surface consistency

One color language.
Four surfaces, in agreement.

The review found the tray, popover, CLI, and statusline disagreeing on when to raise an alarm, and the tray icon contradicting its own label. Both fixes shipped in v0.4.2: a single green / yellow / orange / red scale at 50 / 75 / 90 on every gauge, and a tray whose number can never fight its color. This page is now the record of what shipped and where the surfaces still differ on purpose.

#156 + #157 merged

✓ Shipped · thresholds

Every utilization gauge uses green / yellow / orange / red at 50 / 75 / 90; the statusline greens up below 50. One `Severity` classifier, mirrored to the Svelte popover. #156.

✓ Shipped · tray

Ring, title, and tooltip derive from one view, so a red icon can't sit beside a low number. The tooltip names the worst window and explains cold-start / degraded. #157.
